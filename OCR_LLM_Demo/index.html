OCR + LLM Document Pipeline

Making the complex, easy.
// QUICK CONTEXT

What follows is a ten-module OCR + LLM document pipeline I built in four days as a portfolio piece — originally for an interview at a branded-merchandise / decoration company, but the architecture and patterns generalize to any document-heavy operation.

Click the button below to open the live demo. Each module ingests one document or artwork type and produces structured output ready for downstream systems. Demo data is pre-loaded so you can explore every module without uploading anything yourself, though uploads work too.

A modular OCR + LLM pipeline that augments your existing system of record — it does not replace it. Each module ingests one document or artwork type and hands structured data off to the ERP, MRP, or operational platform of choice (or to a Sheets review buffer the team approves before ingest).

🎨 Artwork & Order Builder
Customer logo → color analysis → recommended decoration method → production sheet → multi-line quote with 2D mockups
🧾 Bills & Invoices
Vendor and freight invoices → extract vendor, line items, totals, payment method; flag fields needing human review
💳 Receipts & Expenses
Field-rep receipt snap → merchant, date, total, tax, tip, category → filed to selected customer + job
📋 Spec Sheets
Manufacturer data sheets → standardized reference table (composition, ratings, mechanical, care, certifications)
✍️ Forms & Notes
Handwritten production-floor notes or printed forms → dates, author, key fields, action items, open questions
🛒 Customer Orders
Emailed order request → customer info, items, decoration spec, in-hands date → filed under customer profile
👥 Customer Library
Browse customers, saved artwork with thumbnails, full order history — the data layer for a future chat-driven ordering assistant
🔧 Universal Text Extractor
Drop any file — PDF, image, SVG, DOCX, AI — and get all text in a copyable block plus contextual notes on what the document is
📊 Reporting
Drill into accepted queues across all modules — by customer, job, category, vendor, date. One view per business question.
📋 Approvals & Change Mgmt
Kanban view of every order by approval stage. Sports/licensed flows (team → league → sponsor) vs. corporate flows. Change orders with audit log.
